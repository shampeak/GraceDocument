## Grace\Easy

## 框架说明
>及简框架
仅仅用作最简单的功能

- 数据库读写
- smarty模板
- 简单路由


## 亮点

- 简单的API
- [RFC3986](http://tools.ietf.org/html/rfc3986) 编码规范
- 组件框架无关[组件可以分离出来单独使用,有独立的文档和测试代码支持]；
- Composer支持
- 遵循[PSR-2](http://www.php-fig.org/psr/psr-2/)和[PSR-4](http://www.php-fig.org/psr/psr-4/)规范
- 单元测试
- 轻量级框架 

## 系统需求

- **PHP >= 5.5.0** , 请使用稳定版本
- `mbstring` 扩展
- `pdo` 扩展
- `curl` 扩展
